# SteamFreeGamesBot
Automated script for read the **free games actually on steam and send post** with the name and link to reclam
Important note⚠️:
*SteamdbFreeGamesBot is a **hobby project** and is not affiliated with Valve, Steam or SteamDB. Steam and the Steam logo are trademarks of Valve Corporation. All other trademarks are property of their respective owners.

* SteamDB/Steam data is not property of this project.

* bot writed with python for read alerts of steamdb freegames and send json that incluide information of the free game avaliable.

## INSTRUCTIONS

* 1. Install python (stable version).
* 2. Install the necesaries librarys with `pip install requirements.txt`.
* 3. Put your discordwebhook url changing the variable of DISCORDWEBHOOK with the valor of 'your-url.'
* EXAMPLE: `DISCORDWEBHOOK = 'Your_url'`.
* 4. Use your github repository and configure a github action for execute the bot x times, you have a screenshot of the parameters of github actions yml.
